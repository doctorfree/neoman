Collection of ascii art fonts for [figlet](http://www.figlet.org/) or [toilet](http://caca.zoy.org/wiki/toilet).

Install files to `/usr/share/figlet/` or `/usr/share/figlet/fonts/`.

These are only the figlet fonts used by `neoman` and are installed in:

`/usr/share/neoman/figlet-fonts/`
